Expansión de historias Walter – Mod para Don't Starve Together

Mod desarrollado para *Don't Starve Together* que amplía el contenido narrativo del personaje **Walter**, agregando nuevas historias y diálogos para enriquecer la experiencia de juego.

El objetivo del proyecto es mejorar la variedad y evitar la repetición de contenido, manteniendo coherencia con el estilo original del juego.

---

Descripción

El mod incorpora nuevas historias para Walter, aumentando la diversidad de diálogos y mejorando la inmersión del jugador.

Las historias fueron diseñadas para integrarse naturalmente con el sistema existente del juego.

---

Tecnologías utilizadas

- LUA
- API de modding de *Don't Starve Together*

---

Características principales

- Nuevas historias y líneas de diálogo para Walter
- Integración con el sistema original del personaje
- Reducción de repetición de contenido
- Expansión narrativa manteniendo coherencia con el juego base

---

Conceptos aplicados

- Extensión de funcionalidades existentes
- Modificación de comportamiento de personajes
- Integración con sistemas preexistentes
- Organización de contenido en archivos LUA

---

Objetivo del proyecto
Mejorar la experiencia del jugador ampliando el contenido narrativo del personaje Walter, aplicando lógica de modificación sobre un sistema ya existente.

---

Cómo usar el mod

1. Copiar el mod en la carpeta correspondiente de *Don't Starve Together*
2. Activarlo desde el menú de mods
3. Jugar con el personaje Walter para ver las nuevas historias
